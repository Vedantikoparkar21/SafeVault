SafeVault is a Document Management System developed using Java, JSP, Servlet, and MySQL.

#Features
- Upload and manage documents
- View and download files
- Convert documents to PDF
- Share via Email and WhatsApp
- Edit document content

# Technologies Used
- Java (Servlets & JSP)
- MySQL Database
- Apache Tomcat Server

# How to Run
1. Import project in Eclipse
2. Run on Tomcat Server
3. Open: http://localhost:8080/SafeVault/
